How we work

ELIXIR is formed by the Member States, each Member State has a Node that provides the services within ELIXIR (see Who we are). These services are coordinated and integrated by the ELIXIR Platforms. Communities use and give feedback on the services, to make sure they are effective in solving real-world problems. All this technical work is driven by a five-year planning cycle - see the current Scientific Programme (2019-2023).
